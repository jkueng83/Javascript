Folgende Aufgaben sind zu lösen:

Passwortschutz

Erweitert eine (oder erstellt eine neue) Webseite mit einem Passwortschutz. Zeigt auf der Startseite einen Login (Username und Passwort Inputfeld) und einen Login Button an. Bei Knopfdruck soll im Javascript geprüft werden, ob es sich um einen gültigen User handelt. Speichert die gültigen User einfach als Konstante im JS. Bei erfolgreichem Login soll der User zur eigentlichen Seite weitergeleitet werden.

Taschenrechner

Erstellt einen Taschenrechner im Web mit den gängigen Rechenoperationen.

Online Verschlüsselungs Tool

Erstelle eine Anwendung die 2 Eingabefelder (textarea) besitzt. Ein Feld für den unverschlüsselten Text & ein Feld für den verschlüsselte Text. Nach der Eingabe soll der Text automatisch verschlüsselt werden. Wird ein Text im 2. Feld eingetragen wird dieser wieder automatisch entschlüsselt. Über ein 3. Eingabefeld soll der “Shift” (um wieviel Buchstaben verschoben werden soll) eingestellt werden können.

Todo App

Erstelle eine App mit der du deine Todo’s verwalten kannst. Über ein Eingabefeld sollen neue Aufgaben erfasst werden können. Diese Aufgaben werden dann in einer List ausgegeben und können mittels Mausklick als erledigt markiert werden. Auch sollen Aufgaben gelöscht werden können.
